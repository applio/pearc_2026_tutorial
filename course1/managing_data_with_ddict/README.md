# Managing Python objects, tensors, and contiguous data with the in-memory distributed dictionary (DDict API)
Data movement between processes/tasks is a key component of AI+HPC workflows and is especially needed
for in situ training and inference. This session introduces the DDict, which provides a transparent data management
layer that spans memory across multiple compute nodes. The goal of this session is to show participants of all levels
how to create a dictionary, control its total memory footprint, and connect it to applications running across nodes.
