# Review/discussion/Q&A/hackathon
This time is provided to allow for open discussion on the topics covered up
to this point. Attendees can use this time to interact with the experts and developers presenting this tutorial and how
the concepts can be applied to their use cases. We will provide a limited set of ideas to drive small group hackathons
for interested attendees, or attendees can invite others to apply the concepts to small challenges of their own. We will
also use this time to gather feedback on how to improve the tutorial and what next steps they may need to further
develop their skills.
