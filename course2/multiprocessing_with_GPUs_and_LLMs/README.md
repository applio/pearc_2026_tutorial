# Using Python multiprocessing with GPUs and PyTorch for multi-node LLM inference
GPUs are a key component in most AI/ML workloads. This session will cover exercises of managing process affinity to GPUs with the
Policy API in DragonHPC, motivated by a distributed LLM inference example. The goal of this session is to show
participants of all levels how to effectively program across multiple GPUs with DragonHPC.
